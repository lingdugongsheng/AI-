📖 项目简介

智能研究助手是一个基于 LangChain 和 LangGraph 构建的自动化研究系统。该系统能够模拟人类研究员的工作流程，自动完成从研究规划、资料收集、信息分析到报告生成的完整研究过程。

系统采用多阶段工作流架构，通过状态化管理确保研究过程的连贯性和可追溯性，特别适用于学术研究、市场调研、技术趋势分析等场景。

✨ 核心功能

🔍 全自动研究流程

系统实现了完整的学术研究工作流，包含6个核心阶段：

研究规划阶段 (Planning)
功能：制定研究计划和大纲
输出：
  研究标题
  研究摘要（100字以内）
  章节结构
  核心研究问题（3-5个）
  研究方法论
智能特性：
  自动识别研究主题的关键维度
  生成符合学术规范的研究框架
  确定需要回答的核心问题

信息收集阶段 (Information Gathering)
功能：多源信息检索
数据源：
  学术数据库：模拟学术论文、期刊文章
  网络资源：新闻报道、行业报告、博客文章
搜索策略：
  精确匹配主题关键词
  相关性评分（0.0-1.0）
  智能降级策略（无精确匹配时返回部分相关结果）
输出：结构化的搜索结果列表

信息分析阶段 (Analysis)
功能：深度信息分析和知识提取
分析维度：
  核心问题的初步回答
  资料中的关键发现
  不同观点的比较分析
  信息空白和研究缺口识别
输出：
  ResearchFinding对象（主题、要点、证据、置信度）
  已分析的资料列表（关键要点、相关性评分）

知识综合阶段 (Synthesis)
功能：整合分析结果，生成报告初稿
综合策略：
  基于研究大纲组织内容
  融合多个来源的观点
  标注引用来源（如 [src_1]）
  保持学术写作风格
输出：各章节的草稿内容（200-300字/章节）

报告生成阶段 (Report Generation)
功能：生成完整的格式化研究报告
报告结构：
  标题
  摘要
  目录
  正文各章节
  结论与建议
  参考文献（标准化格式）
输出：完整的Markdown格式研究报告

质量检查阶段 (Quality Check)
功能：评估报告质量，决定是否迭代
评估维度：
  结构完整性（0-2分）
  论证逻辑性（0-2分）
  引用规范性（0-2分）
  语言表达（0-2分）
  学术严谨性（0-2分）
质量阈值：≥7.5分视为合格
迭代机制：质量不达标时自动重新分析
